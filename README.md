# Student-Performance-Analysis

## Overview
Analysis of students performance in exams using SQL Server and Power BI.

## Dataset
- 1000 students
- 8 columns

## Tools Used
- SQL Server Management Studio (SSMS)
- Power BI

## Analysis
- Gender Distribution
- Race/Ethnicity Distribution
- Average Scores by Subject
- Max and Min Scores by Subject
- Impact of Test Preparation on Scores
- Impact of Parental Education on Scores
- Impact of Lunch Type on Scores
- Pass vs Fail Distribution

## Dashboard
Power BI dashboard with 3 pages:
- Overview
- Scores Analysis
- Factors Analysis

## Key Findings
- 51% of students are female
- Group C is the largest group with 31%
- Math has the lowest average score (66)
- Students who completed test preparation scored higher in all subjects
- Higher parental education leads to better student performance
- Students with standard lunch scored 12 points higher in math
- 81% of students passed all subjects
